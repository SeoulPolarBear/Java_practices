권한 permission

특정 파일에 대해서 읽기/쓰기/실행 여부를 어떻게 할지를 정하는 부분

권한 확인 방법

ls -l을 통해서 확인가능

권한(11개 필드) 하드링크수 소유주 소유그룹 파일크기 수정시간 파일명
- 	      --- 	    --- 			---             .  
:파일의 종류 :소유주  : 소유그룹의 권한	:기타사용자  :acl(접근권한목록) 여부 
-
  일반 파일 (Regular File) 

  d 
  디렉토리 파일 (Directory File)

  b
  블록 디바이스 파일 (Block Device File, (예) /dev/sha, /dev/hda, /dev/fd0)   

  c 
  문자 디바이스 파일 (character Device File. 입출력 장치) 

  l 
  심볼릭 링크 (Symbolic Link File) 


filea에 user01이 접근할 경우
시스템에서 물어봄

당신은 소유자입니까? yes 해당 권한
no -> 당신은 소유그룹에 속해있습니까? yes 해당 권한
no -> 나머지는 모두 기타사용자 권한

권한 우선권 => 소유자 > 소유그룹 > 기타사용자

r : 읽기 권한
w : 쓰기 권한
x : 실행 권한
                  파일                                             디렉토리
r      cat, more/less, 문서편집기를                  디렉토리의 내용 확인
       통한 내용확인                                     (ls, tar, find)
 
w     문서편집기를 통한 내용 수정                  mkdir, touch, mv, rm
                                                              디렉토리의 내용을 수정

x      실행이 가능한 파일 한정으로                  디렉토리에 대한 접근 권한      
       실행 여부

리디렉션 => 화면출력을 하지 않고 파일로 저장하는 것
필요권한 : 파일 생성을 위해 디렉토리의 쓰기
덮어쓰기나 이어쓰기를 하기 위한 파일의 쓰기
