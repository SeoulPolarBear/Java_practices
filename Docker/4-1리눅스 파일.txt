리눅스 파일
모든 것은 파일이라는 철학을 따름
	모든 인터렉션은 파일을 읽고, 쓰는 것처럼 이루어져 있음
	마우스, 키보드와 같은 모든 디바이스 관련된 기술도 팡리과 같이 다루어짐

파일 네임스페이스
	A드라이브 (A:), C 드라이브(C:/windows) (X)
	전역 네임스페이스 사용
		/media/floofy/dave.jpg (/ : root라고 한다.)


쉘 종류
	쉘(shell) : 사용자와 컴퓨터 하드웨어 또는 운영체제간 인터페이스
		사용자의 명령을 해석해서, 커널에 명령을 요청해주는 역할
		관련된 시스템골을 사용해서 프로그래밍이 작성되어 있다.

Linux 쉘 종류
Bourne-Again Shell(bash) : GNU 프로젝트의 일환으로 개발됨, 리눅스 거의 디폴트이다.
Bourne Shell (sh)
C Shell (csh)
Korn Shell (ksh) : 유닉스에서 가장 많이 사용됨

이 때 리눅스 기본 명령어는 기본적으로 bash에서 제공하는 기본 명령어를 배우는 것이라고 보면 된다.

whoami : 로그인한 사용자 ID를 알려줌 / super 관리자 root 웬만해서는 사용하지 않는다.
ps aux : 실행중인 전체 리스트를 출력할 때 사용하는 command
