HTML(Hyper text markup language) =>
hyper text : 링크(link),
markup language : 태그(Tag)를 사용해서 문법을 표기하는 언어
프로그래밍 언어(X) -> 표현언어 (o) : 브라우저 화면에 뭔가를 출력하는 기능만 있는...
Web-site를 만들 때 쓰는 디자인 언어 + 인터프리터방식의 언어(원시 언어의 명령을 번역 실행하는 프로그램.)
컴파일 언어 VS 인터프리터 언어
컴파일러 언어 : JAVA - 컴파일 -> .class 실행 ->
인터프리터 언어 : .html 작업 -실행 -> 한줄씩 기계어로 번역해서 실행
-> 잘못된 상태에서 실행해도 error는 나오지 않음
